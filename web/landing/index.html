01 — WAS DASKALA KANN

Ein Werkzeug für den Notenalltag.

Alles, was ein Notenbuch braucht – ohne Ablenkung, ohne Anmeldung, ohne Datenreise in die Cloud. Ursprünglich für die Mittelschule entwickelt und für jede Schulstufe geeignet.

Noten & Beurteilung

Leistungen erfassen, gewichten und den Notenstand einer Klasse jederzeit im Blick behalten.

Klassen & Schüler:innen

Klassenlisten anlegen und pflegen, mit freundlichen Avataren statt sensibler Fotos.

Jahresplanung

Das Schuljahr strukturieren und Beurteilungen den richtigen Zeiträumen zuordnen.

Export & Import

Ergebnisse als Excel- oder Word-Datei ausgeben und Inhalte aus PDF einlesen.

Läuft überall

Eine installierte Desktop-App für Windows und Linux – kein Browser-Tab, kein Server, keine Abhängigkeit vom WLAN.

Daten bleiben bei dir

Alle Einträge liegen in einer lokalen Datenbank auf deinem Gerät – nichts verlässt den Rechner.

EXTRA · FÜR SCHÜLER:INNEN

Den eigenen Avatar selbst gestalten

Der Avatar-Builder ist eine separate kleine Website für deine Schüler:innen – nicht Teil der App und ohne Anmeldung. Dort stellen sie ihr Gesicht selbst zusammen und erhalten am Ende einen kurzen Code.

Diesen Code schicken sie dir (z. B. über Teams); du gibst ihn in Daskala im Avatar-Editor ein – schon erscheint ihr Wunsch-Avatar in deiner Klassenliste. Es werden dabei keine Fotos, keine Namen und keine Daten übertragen oder gespeichert; der Avatar entsteht nur im Browser.

Avatar-Builder ansehen

Am besten den Link avatar.schulapps.at an die Klasse weitergeben.

02 — LOSLEGEN

Installieren – auf zwei Wegen.

Der einfache Weg für den Alltag, der zweite für alle, die selbst bauen oder mitentwickeln möchten.

Empfohlen

Fertige App installieren

Der schnellste Weg – herunterladen, öffnen, fertig. Verfügbar für Windows und Linux.

  1. Auf der Releases-Seite die neueste Version öffnen.
  2. Die passende Datei laden – Windows: Installer (.exe), Linux: AppImage oder .deb.
  3. Ausführen bzw. installieren – fertig.
Zu den Releases
Plattformen: Windows (.exe-Installer) · Linux (AppImage & .deb).
Updates: Windows und die Linux-AppImage aktualisieren sich selbst; das .deb-Paket aktualisierst du über neue Releases.
Für Entwickler:innen

Aus dem Quellcode bauen

Repository klonen, Abhängigkeiten installieren, starten oder ein eigenes Installationspaket bauen.

$ # Repository holen $ git clone https://github.com/tobGat/daskala.git $ cd daskala $ # Abhängigkeiten (baut better-sqlite3 nativ) $ npm install $ # Entwicklungsmodus: Vite + Electron $ npm run dev $ # Paket fürs eigene System erzeugen: .exe (Windows) bzw. AppImage/.deb (Linux) → dist-electron/ $ npm run build
Voraussetzungen: Node.js & npm. Zum nativen Bauen von better-sqlite3 werden Build-Werkzeuge benötigt – unter Windows die Visual-Studio-Build-Tools, unter Linux build-essential und Python.
03 — HALTUNG

Datenschutz ist kein Feature. Es ist die Grundlage.

  • Schüler:innendaten bleiben lokal auf deinem Gerät gespeichert.
  • Keine Cloud, keine Konten, kein Tracking, kein WLAN nötig.
  • Quelloffen – jede Zeile ist nachvollziehbar und überprüfbar.
